Stage de deuxième année

Sujet :

Notre maitre de stage, James Vallée, est chargé de remplir les feuilles concernant les expulsions locatives.
L'ancienne application étant peu pratique, celui-ci nous a demandé de refaire une application.

Résumé des objectifs :

Création d'une application avec le framework Symfony en php, capable de renseigner les nouvelles expulsions locatives et les expulsions déjà existantes.
Possibilité d'imprimer les feuilles nécessaires après avoir rempli les informations d'une expulsion.
Mise a jour en temps réel grâce à JQuery et Ajax.

Outils utilisés

PHP Storm

Ce projet étant constitué essentiellement de php, j'ai utilisé cet IDE pour le développer. Celui-ci fonctionne parfaitement avec symfony.



JQuery et Ajax

Ce projet nécessitant des mises à jour en temps réel, nous avons dû utiliser ces deux outils.


MySQL

Nous avons utilisé une base de données My SQL pour stocker les différentes données de ce projet.
