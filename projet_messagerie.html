Projet Messagerie

Sujet :

Créer une messagerie dans la console avec le langage Java qui stocke les données dans une base de données.

Résumé des objectifs :

L'objectif était de mettre en œuvre un connecteur permettant de directement agir avec le serveur de la base de données MySQL et ce grâce à l'API JDBC.




Outils utilisés

IntelliJ

Ce projet étant constitué essentiellement de java, j'ai utilisé cet IDE pour le développer.

Git

Ce projet étant un projet de groupe à deux personnes, nous avons utilisé Git.


MySQL

Nous avons utilisé une base de données My SQL pour stocker les différentes données de ce projet.
